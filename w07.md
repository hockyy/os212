## 12.[On the banker's algorithm (Artikel 12')](http://ssomayyajula.github.io/posts/2018-05-18-on-the-bankers-algorithm.html)

Banker's algorithm ialah algoritma untuk **mencegah** terjadinya deadlock! Bukan untuk recovery atau mengobati ya ğŸ’Š! Sebenarnya ada dua masalah utama dalam deadlock problem.

1. Kita tidak tahu kapan suatu proses cukup berhenti menggunakan suatu resource.
2. Kita tidak tahu kapan dia butuh menggunakan resource lain dan akan diam menunggu.

Suatu sistem akan tidak aman bila worstcasenya bisa mencapai deadlock. Banker's algorithm akan melakukan iterasi di mana ada ada sebuah sequence sehingga setiap proses dapat mencapai maksimum resource nya, dan ditutup lagi seperti itu. Untuk lebih jelasnya baca postingannya deh! ğŸ˜­
